Screenshot checklist for APKPure

Take 3-8 phone screenshots (portrait, 1080x1920 or device native).
Suggested screens:
1. Dashboard overview
2. Add Trade form (with data filled)
3. Analytics view
4. Review/Replay view with screenshots
5. Settings

Tips:
- Use real data and clear visuals.
- Avoid notifications in the status bar.
- Keep consistent theme (dark or light) across the set.

Save files in this folder and name them like:
